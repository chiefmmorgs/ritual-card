Ritualist ID Card Generator

Deliverable
One file: index.html with embedded CSS and vanilla JS. No external libs. Runs locally. Exports a crisp PNG.

Goal
Build a single-page site that lets a user:

enter name

choose two roles: Core Role and Aura Role

auto-include the Encore Role “Pledged” on every card

pick Date of Join

upload a photo (center-crop to a circle avatar)

live-preview the ID card

download the result as a PNG at 2× scale

Brand
Monochrome ritual theme. Gritty grayscale, high contrast white text, faint film grain. 


Roles
Core roles (select one):

mage

ritualist

ritualist ascendant

node runner

Aura roles (select one):

initiate

blessed

cursed

harmonic

Encore role (always present, not selectable):

pledged

Palettes
Core base gradients

Aura tints and effects

blessed: tints #FFD166 → #FFF7E6, opacity 0.28, blend soft-light, add gentle bloom

cursed: tints #00FF95 and #7A2DFF, opacity 0.42, blend color-burn, add vignette and light chromatic fray

harmonic: tints #66E3FF → #BD7CFF, opacity 0.34, blend screen, add subtle RGB split

Encore layer

pledged: always on

chip color #B7FF3C, pulse scale 0.98–1.02 over 4 s

faint binding lines using a low-opacity noise mask

Form controls
Inputs under the canvas in a rounded toolbar:

Name [text] placeholder “e.g. Elliott Yu”

Core Role [select] values: mage, ritualist, ritualist ascendant, node runner

Aura Role [select] values: initiate, blessed, cursed, harmonic

Portfolio [text] placeholder “e.g. Educator” (optional)

Date of Join [date] shown on card as YYYY-MM-DD

Upload Photo [file] accept images; center-crop to square; mask to circle

Download PNG [button]

Reset [text link]

Export 2× sheet [text link] optional

Behavior

Any input change updates the preview instantly

Date displayed as YYYY-MM-DD regardless of locale

Core + Aura drive the theme; Pledged always appended in the Role line and chip layer

Node Runner adds grid + scanline sweep every 6 s at 6 percent opacity

Download exports a PNG named ritualist-id-card.png, 2× scale, sharp text

Optional Export 2× sheet renders four variants of the same data:

Core+current Aura

Core+blessed

Core+cursed

Core+harmonic

Contrast AA or better

JS modules inside a single <script>:

optional exportSheet()
